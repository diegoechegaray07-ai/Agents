# Diseño visual — Informe de Ventas Getnet

> Ya implementado en `scripts/generar_informe.py`. Leé esto solo si vas a
> **modificar** el diseño; para generar el informe no hace falta.

**Paleta:** azules pasteles
- Fondo portada: `#0D3B6E` (azul oscuro)
- Acento: `#4DA3E0` (celeste)
- Header tablas: `#1B4F8A`
- Fondo KPI: `#EBF4FF`
- Filas alternas: `#EBF4FF` / blanco
- Total: `#D0E8FB`

**Encabezado:** banda lateral celeste (`0.55cm`) + bloque de texto sobre fondo oscuro
→ "INFORME DE VENTAS" (pequeño, espaciado) / Nombre propietario (grande) / Período

**KPIs:** cada tarjeta con header azul oscuro (label) + cuerpo con línea inferior celeste

**Títulos de sección:** barra lateral celeste + texto sobre fondo suave `#F0F7FF`

**Sin gráficos. Sin sección de criterios aplicados.**
